# Proyecto Rico Pío - Sistema de Facturación

Este proyecto es un sistema de gestión para el restaurante "Rico Pío", desarrollado como parte de un proceso de aprendizaje en Ingeniería de Sistemas y bajo la filosofía de "Centauro Profesional".

## Estructura del Proyecto

El proyecto sigue una organización modular para facilitar el mantenimiento y escalabilidad:

- 📂 `src/`: Contiene el código fuente de la aplicación (`RicoPioApp.pyw`).
- 📂 `data/`: Almacenamiento de bases de datos y archivos de configuración.
- 📂 `docs/`: Documentación técnica, requerimientos y manuales de usuario.
- 📂 `assets/`: Recursos visuales como iconos, logotipos e imágenes.
- 📂 `output/`: Carpeta de salida para facturas y reportes generados.
- 📂 `tests/`: Scripts de prueba para verificar la funcionalidad del sistema.

## Ejecución

Para iniciar la aplicación, simplemente ejecuta el archivo lanzador en la raíz:

```bash
python main.py
```
